# Keepy-Uppy King

A self-contained retro pixel-style browser mini-game.

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Controls

- Tap / click / spacebar: kick
- Left/right arrows or A/D: help the player shuffle

## Features

- Perfect timing bonus
- Combo multiplier
- Increasing difficulty
- Daily challenge mode
- Unlockable ball skins with local coins
- Best score stored in localStorage
- Share/copy score
- Retro pixelated canvas rendering
